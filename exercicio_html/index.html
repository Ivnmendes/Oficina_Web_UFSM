Temas do meu interesse

Desenvolvimento web: Desenvolvimento web é o processo de criação, codificação e programação de sites, aplicativos e ambientes de intranet. O profissional que trabalha nessa área é chamado de desenvolvedor web, web developer ou programador.

Desenvolvimento back end: Desenvolvimento back-end é a área de programação que se dedica à parte interna de uma aplicação, ou seja, à estrutura e aos processos que fazem com que a aplicação funcione.
